Auto Writing Notepad

per quanto il nome possa ingannare, questo programma è completamente inutile.
è stato creato il 1 gennaio 2022 e risulta essere il mio secondo programma mai rilasciato al pubblico
esso è un programma scherzo che una volta aperto non fa altro che scrivere "il tuo computer è hackerato" e creare una finta RSOD

Questo programma è archiviato permanentemente



AddPlayersMC

questo programma è una mod per il popolare gioco Minecraft e dà la possibilità al giocatore di aggiungere NPC (Alex e Steve) al gioco
inizialmente creata con Mcreator e successivamente continuata con IntelliJ Idea Community edition la mod è sia scaricabile per il toolkit Forge e Fabric

Ultima versione compatibile con Minecraft 1.17.1

Nget

Nget è un programma scritto in Node.JS che permette di scaricare file da internet sia da riga di comando che da interfaccia web

per installare il programma è necessario avere Node.JS e NPM installato sul proprio computer, dopodiché è sufficiente eseguire il comando npm install -g @star08-w3b/nget
il pacchetto è compatibile con altri package manager che utilizzano il registro di NPM
